# Ghazara - دليل المشروع الشامل

## نظرة عامة على المشروع

مشروع **Ghazara Landing Page** هو موقع هبوط احترافي لشركة متخصصة في التسويق الرقمي. تم تصميم الموقع باستخدام أحدث التقنيات الحديثة (React 19 + Tailwind CSS 4 + Vite) لضمان أداء عالي وتجربة مستخدم ممتازة.

**النطاق**: ghazara.net  
**منصة النشر**: Vercel  
**موفر النطاق**: Hostinger  
**الإطار العمل**: React + Vite  
**نظام التصميم**: Tailwind CSS 4

---

## فلسفة التصميم

تم اختيار **Modern Minimalist with Bold Accents** كفلسفة التصميم الأساسية. هذا النمط يجمع بين:

- **البساطة والوضوح**: تصميم نظيف يركز على المحتوى
- **الاحترافية**: ألوان وتصميمات تعكس خبرة الشركة
- **المرونة**: سهولة التوسع والتطوير المستقبلي
- **التفاعلية**: حركات وتأثيرات ناعمة وأنيقة

### نظام الألوان

## خطوات البدء السريعة

### 1. تثبيت المتطلبات

```bash
# تحقق من إصدارات Node.js و npm
node --version
npm --version

# أو استخدم pnpm (موصى به)
pnpm --version
```

### 2. تثبيت المكتبات

```bash
# استخدام pnpm (الموصى به)
pnpm install

# أو استخدام npm
npm install
```

### 3. تشغيل خادم التطوير

```bash
# استخدام pnpm
pnpm dev

# أو استخدام npm
npm run dev
```

سيتم تشغيل الموقع على `http://localhost:3000`

### 4. بناء المشروع للإنتاج

```bash
# استخدام pnpm
pnpm build

# أو استخدام npm
npm run build
```

## الأداء والتحسينات

### معايير الأداء المستهدفة

- **Lighthouse Score**: > 90
- **Core Web Vitals**: جميع المؤشرات خضراء
- **وقت التحميل**: < 3 ثوانٍ
- **حجم الصفحة**: < 500 KB

### تحسينات الأداء

1. **ضغط الصور**: استخدم أدوات مثل TinyPNG
2. **التخزين المؤقت**: فعّل HTTP caching
3. **CDN**: استخدم CDN للأصول الثابتة
4. **Code Splitting**: قسّم الكود إلى أجزاء أصغر
5. **Lazy Loading**: حمّل الصور والمكونات عند الحاجة

## استكشاف الأخطاء الشائعة

### المشكلة: الموقع لا يعمل محلياً

**الحل:**
```bash
# حذف node_modules وإعادة التثبيت
rm -rf node_modules
pnpm install

# أو استخدم npm
npm install
```

### المشكلة: الأنماط لا تظهر بشكل صحيح

**الحل:**
1. تأكد من أن Tailwind CSS مثبت بشكل صحيح
2. تحقق من ملف `tailwind.config.js`
3. أعد تشغيل خادم التطوير

### المشكلة: النطاق لا يعمل بعد النشر

**الحل:**
1. تحقق من سجلات DNS في Hostinger
2. انتظر 24-48 ساعة لانتشار DNS
3. امسح ذاكرة التخزين المؤقت للمتصفح
